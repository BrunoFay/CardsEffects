# Card Effects
foi utilizado os métodos de animação, transform e box-shadow.

### Rotação
* transform: rotate

### Rotação X
* transform: rotateY

### Rotação Y
* transform: rotateX

### Flip 3D
* transform: skewX
* transform: skewY

### Zoom
* transform: scale

### Movimento top
* transform: translateY

### Movimento 3D
* transform: rotateZ
* transform: rotateX
* transform: rotateY
* transform: scale
